# rogue

Rogue 통합 레포 — 문서 + 프로젝트 코드.

## 구조

- docs/ — 조직 운영 문서
- research/ — 리서치, 시장 분석
- resources/ — 첨부 자료, 참고 이미지, PDF
- team/ — 팀 관련 문서
- projects/ — 프로젝트별 폴더 (문서 + 코드 통합)
  - medical/ — 병원 마케팅 (제안서, 크롤링, 리서치, 웹사이트)
  - moutai/ — 마우타이 관련

## 규칙

- 폴더명은 영어, 문서 내용은 한국어로 작성
- 새 문서는 해당 폴더에 생성
- 참고 자료는 resources/에 저장
- 프로젝트별 CLAUDE.md가 각 프로젝트 폴더에 존재
