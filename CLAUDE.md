# rogue-docs

Rogue Docs — git 기반 팀 문서 + 프로젝트 코드.

## 구조

- 운영/ — 조직 운영 문서
- 리서치/ — 리서치, 시장 분석
- 자료/ — 첨부 자료, 참고 이미지, PDF
- 미팅/ — 모든 미팅 기록 (파일명에 프로젝트명 포함: `날짜-프로젝트-주제.md`)
- 팀/ — 팀 관련 문서
- 프로젝트/ — 프로젝트별 폴더 (문서 + 코드 통합)
  - 병원마케팅/ — 병원 마케팅 (제안서, 크롤링, 리서치, 웹사이트)
  - 마오타이/ — 마오타이 관련
  - 업무환경구축/ — 업무 환경 구축 (일하는 방식, 작업 환경, 활용 솔루션)

## 규칙

- 폴더명, 문서 내용 모두 한국어로 작성
- 새 문서는 해당 폴더에 생성
- 참고 자료는 자료/에 저장
- 프로젝트별 CLAUDE.md가 각 프로젝트 폴더에 존재

## 시스템 문서 유지

- 시스템/ 폴더는 팀 로그의 운영 시스템 문서로, 항상 최신 상태를 유지해야 한다
- 폴더 구조 변경, 프로젝트 생성, 규칙 변경 등이 있을 때 해당 시스템 문서를 함께 업데이트한다
