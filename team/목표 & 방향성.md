## 진행 중인 꼭지

### 1. 의료 마케팅 대행

→ [[문서/의료 PoC]] 참고

### 2. 행정사

- 외국인 비율이 높아짐에 따라 비자 행정 대행 수요 증가
- 외국인 입장에서 신뢰가 어려움, 행정사 인재 밀도가 낮아서 차별화 여지
- 전략: 마케팅 대행 → 업무 자동화 → 직접 인수해서 하기

### 3. 인수창업 (VC 포폴)

- 전략: VC 포폴 중 하방 막힌/상방 열린 스타트업 인수
- 검토 중:
	- 허밍버드(아기고래) — HR 슬랙 인앱, MRR 500만원, 매각 희망 (프라이머 투자금 반환 예정)
	  - 대표 27세, 프라이머 2억 투자받았으나 미사용 → 반환 예정
	  - 고객사 30곳, 대형고객 강남언니 85만원 + 핸디스/유닛블랙/페이히어/딥다이브 각 40만원
	  - 순수익 ~400만원/월, 예상 매각가 5천~1억
	  - 창업자는 매각 후 대기업 취업 희망
- 참고: 인재 인수 케이스만 매력적일 수 있음

### 4. 인수창업 (Top-down 리서치)

- 시장 리서치로 매물 직접 탐색
- 검토 중:
  - 새한A&G — 전자경매시스템
  - 보강시스템 — 섬유 공장 ERP

### 5. 패션 커머스

- 동대문 소싱 기반 커머스
- 검토 중: 순이익 월 2,000만원, 매매가 1.5억 매물 (회수기간 8개월)
- 리스크: 운영 난이도 높음, 숫자가 너무 좋아서 의심 필요
- 운영 실체: 매일 1위 업체 사이트 돌면서 상품 서칭 → 사입삼촌에 샘플 주문 → 사진 찍고 상세페이지 수정 → 마케팅 → 주문 들어오면 건건이 발주

### 6. 렌터카 ERP

- 플러스 렌터카 ERP 연락 예정
