- `3-noti-` : 알림 (자동화)

## 자동화

### 문서 변경 → Slack 알림

- **트리거**: main 브랜치에 push 시 문서 폴더 변경 감지
- **채널**: `#3-noti-github`
- **내용**: 변경 파일 목록 + 커밋 메시지
- **구현**: [GitHub Actions 워크플로우](../.github/workflows/slack-notify.yml)
- **설정 필요**: GitHub Secrets에 `SLACK_WEBHOOK_URL` 추가

### 설정 방법

1. Slack 앱 > Incoming Webhooks 활성화
2. `#3-noti-github` 채널에 Webhook 추가
3. Webhook URL 복사
4. GitHub repo > Settings > Secrets > `SLACK_WEBHOOK_URL`로 추가
