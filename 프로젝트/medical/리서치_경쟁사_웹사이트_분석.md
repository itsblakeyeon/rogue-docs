- **메인 헤드라인**: "1위가 관리해야 1위가 됩니다"
- **서브카피**: "전국 2,000여 원장님들이 선택한 믿을 수 있는 당신의 든든한 병원마케팅 파트너"

- **500+** 관리 병원
- **13년** 업력
- **98%** 연장률 / 99% 재계약률
- **2,000곳** 이상 누적 파트너
- **11관왕** 수상

- 블로그 방문자 **5.14배** 증가
- 원장님 출연 영상 후기 **500개** 이상

- **다크 배경** (진한 네이비/블랙) + 흰색 텍스트

- 핵심 숫자를 아이콘과 함께 히어로 하단에 배치 (500+ / 13년 / 98%)
- 깔끔한 산세리프 폰트, 대형 볼드체 헤드라인
- CTA 버튼: 흰색 배경 + 화살표 아이콘 ("무료 상담 신청 ->")

### 제안서 참고 포인트
- **숫자 나열 방식이 매우 효과적** - 히어로 영역 하단에 3개 핵심 지표를 아이콘과 함께 배치
- "1위가 관리해야 1위가 됩니다" 같은 **자신감 있는 포지셔닝** 문구
- AI/기술력 강조 ("자체 AI 개발팀 보유, 업계 유일")
- 원장님의 시간을 아껴준다는 **공감 메시지**

## 6. 상승기획 (Sangseung) - sangseung.net

### 핵심 메시지
- **메인 헤드라인**: "모든 병원의 마케팅을 맡지 않습니다." (SELECTIVE CONSULTING)
- **서브**: "저희는 성과를 만들 수 없다고 판단되면 처음부터 진행하지 않습니다."
- **핵심 약속**: "효과가 없으면 상승은 [환불]하기 때문입니다."
- **반복 메시지**: "상승은 매출상승이라는 결과로 증명합니다"

### 신뢰 신호 (매우 공격적)
- **156개** 병원 1년 이상 협업
- 전체 평균 **358%** 매출 상승
- **96%** 광고 만족도
- **93.7%** 재계약률
- 평균 검색 유입 **258%** 상승
- 예약 검색률 **153%** 상승
- 신환 **300%** 증가 사례
- 유튜브 구독자 **21만** 채널 보유
- 원장님 출연 **영상 후기** 다수 (YouTube 임베드)

### 서비스 구조
- 올인원마케팅 (블로그, 플레이스, 인스타, 유튜브, 스레드, 홈페이지)
- 외국인환자유치
- 병원 홈페이지 제작
- 병원컨설팅
- 텔레마케팅

### 차별화 전략: "스토리마케팅"
- "환자는 설득됐을 때 지갑을 열기 때문입니다"
- 무작정 글 쓰기가 아닌 스토리 기반 콘텐츠 마케팅

### 비주얼 디자인
- **다크 배경** + 핑크/레드 포인트 컬러
- 매우 공격적인 팝업 (유튜브 출연 유도 + 문의 유도)
- YouTube 영상 후기를 **대량으로** 배치 (6~8개 영상)
- 각 성과를 그래프/차트 이미지로 시각화
- 실제 원장님 차트 캡처 등 **증거 자료** 직접 노출
- 하단 고정 플로팅 버튼: 카톡상담 + 빠른문의
- Q&A 아코디언 섹션
- **"환불 보장"** 메시지를 마지막 CTA에 강력 배치

### 페이지 구조 (순서)
1. 히어로: 선별 상담 메시지 + CTA
2. 환불 보장 메시지
3. 자신감의 이유 (영상 후기)
4. 협업 병원 성과 그래프 (358% 평균)
5. 만족도/재계약률 숫자
6. 3단계 성과 증명 (검색 유입 -> 예약 -> 신환)
7. 실제 후기 이미지
8. 추가 정보 카드 (스토리마케팅/진행과정/서비스)
9. Q&A
10. 환불 보장 CTA

### 제안서 참고 포인트
- **"안 되면 환불"** - 극단적 리스크 제거 메시지가 강력
- **"모든 병원을 맡지 않는다"** - 선별적 접근이 오히려 프리미엄 느낌
- 영상 후기의 **물량 공세** - 신뢰감 극대화
- 3단계 성과 플로우 (유입 -> 예약 -> 신환)가 **퍼널 개념**을 쉽게 전달
- 실제 데이터 캡처를 그대로 보여주는 **증거 기반 접근**

## 종합 분석 및 제안서 적용 인사이트

### 1. 공통 디자인 패턴

| 요소 | 패턴 |
|------|------|
| **배경** | 다크 테마(블랙/네이비) 압도적으로 많음 - 프리미엄 느낌 |
| **히어로** | 풀폭 이미지/영상 + 오버레이 + 대형 헤드라인 |
| **숫자** | 히어로 하단에 3~4개 핵심 KPI 아이콘과 함께 나열 |
| **CTA** | "무료 상담 신청" / "문의하기" + 카카오톡 연동 |
| **후기** | 인용문 or 영상 후기 (원장님 직함 포함) |
| **플로팅** | 카톡 + 전화/문의 하단 고정 버튼 |

### 3. 우리 제안서에 적용할 핵심 요소

#### 반드시 넣을 것
1. **숫자 기반 신뢰**: 하룹/상승처럼 핵심 성과 지표 3~4개를 시각적으로 배치
   - 화이트큐브 0->280억, CAC 50% 감소, ROAS 174%p 상승
2. **프리미엄 포지셔닝**: 진저처럼 "모든 곳을 받지 않는다" / "소수 파트너와 깊게"
3. **프로세스 시각화**: 인지->상담->예약->내원 퍼널을 테라미디어/상승처럼 단계별로
4. **고객 공감 섹션**: "원장님의 고민" - 모든 사이트가 공감 -> 해결 구조

#### 차별화 포인트 (경쟁사에 없는 것)
1. **매출까지 추적** - 대행사들은 "유치해 드립니다"까지만, ROI 추적은 언급 없음
2. **데이터 대시보드** - 메디빌더만 유사, 나머지는 기술 요소 없음
3. **AI 자동화** (챗봇, 예약) - 메디빌더가 개발 중이나 아직 미출시
4. **실제 오프라인 사업 성장 경험** - 대부분 마케팅 대행사 출신, 우리는 사업 운영자 출신

#### 디자인 방향
- **다크 테마 + 포인트 컬러** (블루 or 그린 계열로 차별화)
- 히어로: 한 줄 태그라인 + 서브카피 + CTA + 핵심 숫자 3개
- 서비스 카드: 아이콘 + 제목 + 설명 (진저 스타일)
- 성과 섹션: 그래프/차트 + 실제 수치 (상승 스타일)
- 팀 소개: 실적을 문제->실행->결과 구조로 (CLAUDE.md 방향 유지)

#### CTA 전략
- 경쟁사: "무료 상담 신청" / "문의하기" / "제안서 요청"
- 우리: **"무료 진단 리포트 받기"** 등으로 차별화 (데이터 기반 이미지 강화)
