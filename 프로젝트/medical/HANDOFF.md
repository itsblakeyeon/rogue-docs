# Handoff

## 목표
병원 대상 비즈니스 제안서 (영업용, HTML 15슬라이드) 완성 및 영업 진행.

## 현재 상태
- 제안서 HTML 15슬라이드 완성 (콘텐츠 + 디자인)
- 팀 킬러넘버 "57억→280억" (5배 성장)으로 확정
- 리서치 문서 4개 완성 (국내마케팅, 해외마케팅, 시장분석, 팀분석, 경쟁사)
- 아직 커밋/푸시 안 됨

## 완료 사항

### 제안서 슬라이드 (15개)
1. 표지 — "데이터 기반 병원 그로스 파트너, 로그"
2. 팀 킬러넘버 — 57억→280억 (5배 성장)
3. 팀 프로필 — 코어 2명 + 서포트 2명 (LinkedIn 뱃지 이름 옆)
4. 시장 기회 — FOMO ("경쟁 병원은 이미 움직이고 있습니다")
5. 병원의 문제
6. 기존 업체 한계
7. 우리의 차이 — ①성과 측정 ②사업 이해도 ③운영방식
8. 이렇게 일합니다 — 퍼널 + AI/자동화
9. 국내 마케팅
10. 해외 마케팅
11. 프로세스 (6개월 타임라인)
12. CTA — "무료 1시간 상담" + 구글 폼
13. 어펜딕스 1 — 국내 시장 데이터
14. 어펜딕스 2 — 해외 시장 데이터 (출처 링크)
15. 어펜딕스 3 — 병원 마케팅 페인포인트

### 세션별 주요 변경
- 브랜딩: "ROGUE 로그" → "데이터 기반 병원 그로스 파트너, 로그"
- 4페이지: VC 언어("시장이 크다") → FOMO("경쟁 병원은 이미 움직이고 있습니다")
- 7페이지: "국내+해외 범위" → "사업 이해도"로 차별화 교체
- CTA: 시장 규모 숫자 → 페인포인트 질문
- 어펜딕스 3 신규 추가
- 팀 킬러넘버: 147억 → 280억 복원

## 미완료
- **커밋/푸시**: 모든 변경사항 unstaged
- **구글 폼 응답 알림**: 폼 생성 완료, 이메일 알림 미설정
- **PDF/PPT 변환**: 최종 포맷 미정

## 결정 사항
- **57억→280억 확정**: 사용자가 280억(원래 값)으로 복원 요청하여 확정
- **브랜딩**: "데이터 기반 병원 그로스 파트너, 로그"
- **피부과/성형외과 특정 금지**: "병원"으로 일반화
- **팀 사진 불포함**: 프로페셔널하지 않다는 이유

## 주의 사항
- **팀 프로필 규칙**: 이모지 없음, 영문이름 없음, 스킬 나열 없음, 대학 인라인, LinkedIn 뱃지(이름 옆)
- **김영욱 역할**: "그로스 마케팅 + 프로덕트 빌딩" (퍼포먼스 마케팅 아님)
- **이윤규 제페토**: "네이버 제페토"로 표기
- **CTA**: "무료 1시간 상담" (진단 리포트 아님)
