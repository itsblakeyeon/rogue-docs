# What
- 최종 결과물: 영업용 비즈니스 제안서 PDF (병원 원장 대상, 범용)
    - 수정이 가능하도록 PPT나 피그마로 먼저 만들 것.
    - What보다 How를 강조하는 방향. "우리는 이렇게 일합니다 — 감이 아니라 숫자로."

# Why
- 창업 아이템으로 버티컬 AI 비즈니스를 보는 중, 우선은 의료 산업에 집중.
- 외국인 의료관광 폭증 (2024년 117만→2025년 160만, 피부과 56.6%) + 정부 적극 지원.
- 기존 대행사/에이전시는 성과 불투명, 매출 연결 추적 안함 → 데이터 기반 성장 파트너 포지셔닝 기회.

# Who (타겟 고객)
- **중견 피부과/성형외과** (의사 3~5명, 직원 20~50명, 월매출 5~10억)
- 이미 대행사를 쓰고 있을 확률 높음. 외국인 비중을 늘리고 싶은 단계.
- 1인 개원의는 지불 여력 부족, 대형 프랜차이즈는 자체 팀 보유 → 중견이 스윗스팟.

# How

## 포지셔닝
- **"매출까지 보는 외국인 환자 성장 파트너"**
- 대행사(클릭만 봄)도 아니고, 에이전시(컨시어지만 함)도 아닌 제3의 포지션.
- 차별화: 전환 퍼널 전체(인지→상담→예약→내원)를 데이터로 추적, 매출까지 연결.

## 서비스 2축
- **A축 (외국인 환자 유치 — 디지털 마케팅)** ← 1단계, 먼저 진입
    - 국적별 해외 SNS 운영 (샤오홍슈, 인스타, 구글 등)
    - 퍼포먼스 광고 (국적별 채널 최적화)
    - 인플루언서 섭외/협업
    - 전환 퍼널 추적 (어떤 채널 → 몇 명 문의 → 몇 명 예약 → ROI)
- **B축 (운영 효율화 — AI 기반)** ← 2단계, 신뢰 확보 후 확장
    - 다국어 AI 챗봇 (상담 자동화)
    - 예약 시스템 자동화
    - 데이터 대시보드

## GTM 전략
```
1단계: 외국인 환자 유치 마케팅 (매출 성장 증명)
  ↓ 신뢰 구축
2단계: 운영 효율화 (상담/예약 자동화)
  ↓ Lock-in
3단계: SaaS 제품화 (대시보드, CRM, 자동화)
```
- 마케팅이 먼저. 고객들은 비용 줄이는 것보다 매출 늘리는 걸 먼저 생각.
- A축에서 외국인 환자 문의 증가 → 다국어 상담 부담 → B축 자연스럽게 연결.

## 가치제안 (팀)
- 화이트큐브(패러데이)를 함께 0에서 연매출 280억까지 성장시킨 팀.
- 연준현: 그로스 마케팅 + 프로덕트 빌딩 (CAC 50%↓, ROAS 174%p↑, MVP 3일 출시)
- 신지훈: 세일즈 + 사업개발 + 재무 (MRR 0→6억, 200억 조달, VC 심사역 출신)
- 오프라인 비즈니스(자동차 구독, 햄버거 직영점)에서 실제 매출 성장 검증.
- "숫자로 증명합니다" — 모든 프로젝트에 문제→실행→결과(숫자) 구조.

## 제안서 목차
1. 표지 — 회사명, 한 줄 태그라인
2. 시장 상황 — 외국인 환자 폭증, 피부과 56.6%, 정부 지원
3. 원장님의 고민 — 외국인 못 잡음, 대행사 불신, 데이터 없음
4. 기존 솔루션의 한계 — 대행사는 클릭만, 에이전시는 수수료만, 둘 다 ROI 안 봄
5. 우리의 접근 (How) — 데이터 기반 성장 파트너, 매출까지 추적
6. 서비스 A: 외국인 유치 마케팅 — 채널별 전략, 인플루언서, 전환 추적
7. 서비스 B: AI 운영 효율화 — 다국어 챗봇, 예약 자동화
8. 팀 소개 — 화이트큐브 실적, 오프라인 비즈니스 성장 사례
9. 어펜딕스 — 시장 데이터 상세, 레퍼런스
