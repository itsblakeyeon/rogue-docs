## 리서치 (미컨택)

### 2. 패스트파이브 (FASTFIVE)

- **웹사이트**: [fastfive.co.kr](https://fastfive.co.kr/)
- **성수 지점**: 서울숲점, 성수1호점 (2개 지점)
- **위치**: 성수역/뚝섬역 도보 5분
- **2인실 예상 가격**: 월 50~80만 원 (지역별 상이)
- **특징**: 냉난방 개별 컨트롤 가능, 회의실/라운지/주방 등 부대시설 우수, 국내 1위 공유오피스 브랜드

### 2. 스파크플러스 (Spark Plus)

- **웹사이트**: [sparkplus.co](https://www.sparkplus.co/)
- **연락처**: 전화 통화 완료 (2026-03-16)
- **성수**: 2인 입주 불가 (4인실부터 가능)

#### 2인실 가능 지점 견적 (2026-03-16 통화)

| 지점 | 무약정 월세 | 12개월 계약 | 비고 |
|------|------------|------------|------|
| **삼성역점** | 140만 원 (VAT 별도) | 100만 원/월 | |
| **동대문역사문화공원역점** | 132만 원 (VAT 별도) | 90만 원/월 | |

- **정확한 할인율은 지점 방문 후 매니저와 협의 필요**

### 4. 헤이그라운드 (Heyground)

- **웹사이트**: [heyground.com](https://heyground.com/)
- **위치**: 성수 시작점 - 뚝섬역 도보 5분, 서울숲역 도보 6분
- **특징**: 소셜 임팩트 커뮤니티 지향, 스타트업/소셜벤처 중심
- **비고**: 2인실 가격 확인 필요

### 5. 아이엘 공유오피스 (IL Cowork)

- **웹사이트**: [ilcowork.com](https://www.ilcowork.com/)
- **위치**: 뚝섬역 도보 2분 (초역세권)
- **특징**: 호텔식 프리미엄 독립룸, 프라이빗 오피스, 최소 1개월 유연한 계약
- **비고**: 2인실 가격 확인 필요

### 6. 플레이스캠프 성수 (Playce Camp Seongsu)

- **웹사이트**: [playcecampseongsu.com](https://playcecampseongsu.com/)
- **특징**: 성수동 대형 복합문화공간 내 공유오피스
- **비고**: 2인실 가격 확인 필요

### 7. 스테이션니오 (Station Neo)

- **웹사이트**: [stationneo.com](https://www.stationneo.com/)
- **위치**: 성수역 도보 0분 (초역세권)
- **특징**: 코워킹스페이스
- **비고**: 2인실 가격 확인 필요
