# Handoff

## 목표
병원 대상 데이터 기반 마케팅 비즈니스: 크롤링으로 이메일 수집 → 제안서 발송 → 영업.

## 현재 상태

### 제안서 (완료)
- HTML 15슬라이드 완성 (콘텐츠 + 디자인)
- 팀 킬러넘버 "57억→280억" (5배 성장)으로 확정
- 리서치 문서 5개 완성

### 크롤링 (완료)
- HIRA 파이프라인 1~5순위 전체 완료
- 네이버 지역검색 파이프라인 완료
- **최종 수치 (2026-03-19)**: 총 14,196건, 유효 이메일 1,175건 (HIRA 583 + 네이버 592)
- 지역: 서울 988건, 경기 174건

### 웹사이트 (완료)
- therogues.xyz 배포

## 미완료

### 제안서
- PDF/PPT 변환: 최종 포맷 미정
- 구글 폼 응답 알림 미설정

### 크롤링
- `sido` 컬럼 정규화 ("서울특별시" → "서울") 미처리
- 이메일 있는 병원만 HIRA 상세정보 보강 미구현
- 경기도 나머지 25개 시/군 동단위 세부수집 미확대

## 결정 사항
- **57억→280억 확정**: 사용자가 280억(원래 값)으로 복원 요청
- **브랜딩**: "데이터 기반 병원 그로스 파트너, 로그"
- **피부과/성형외과 특정 금지**: "병원"으로 일반화
- **HIRA + 네이버 하이브리드**: HIRA 치과/경기도 데이터 부실 → 네이버로 보완
- **skip-detail 우선**: HIRA API 일일 1,000건 제한 때문에 상세정보 생략
